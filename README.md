# Flow Board

Kanban-Board-App für persönliches Solo-Projekt-Management. Next.js App Router, TypeScript strict, Supabase (Postgres + RLS + Auth), Tailwind v4, shadcn/ui, Motion, dnd-kit.

## Setup

    npm install
    npm run dev
